Jogo de Forca:

- O usuário deverá se identificar
	> inserir seu nome

- O sistema irá saudar o usuário e informar como inicia o jogo.
	> sugestão: "Bem-vindo [Fulano].  Vamos jogar FORCA?
			Você escolhe o tema, eu escolho a palavra.
			Vamos começar!"

- o jogo iniciará informando os temas disponíveis. (vide arquivo temas)

- o usuário escolherá seu tema.

- o sistema informará que já escolheu a palavra dentro do tema e informará quantas letras ela tem.

- Objetivos do jogo:
	Acertar a palavra escolhida pelo sistema através de palpites das letras que compoem a palavra.
		O jogador será "eliminado" se errar seis letras da palavra.
		Se ele conseguir através dos palpites acertar todas as letras para formar a palavra ele vencerá.
			Existe a possibilidade de, com algumas letras já posicionadas mediante acerto, o jogador conseguir adivinhar toda a palavra.
		 	para isso, disponibilize um "botão" que permita que o jogador chute a palavra completa.

- Ações:
	1 - Chutar 1 letra
	2 - Tentar adivinhar a palavra toda
